# FitBot - Premium Wellness & Fitness Coach

FitBot is a premium, wellness-focused coaching application designed with a calming **Sage Leaf + Soft Peach** aesthetic. It moves away from generic chatbot interfaces to provide a personalized, human-centered coaching experience.

## Key Features
- **Intelligent Wellness Coach**: Multi-turn AI conversational partner powered by a local RAG engine.
- **Diet & Nutrition Planner**: View macros and calorie budgets across customizable meal frames.
- **Workout Split Planner**: Interactive routine checklist with built-in stopwatch/timers.
- **Progress Telemetry**: Visual line charts tracking weight trends, BMI classification, and calorie analytics.
- **Interactive Visuals**: Calming botanical margins with cursor-responsive gradient blobs that follow mouse movement with smooth, slow easing.
- **Supabase Authentication**: Secure email registration, Google OAuth, and password recovery redirection.

## Architecture
- **Frontend**: React, Vite, Tailwind CSS, Lucide icons.
- **Backend**: Node.js, Express, Supabase client, local vector knowledge base.

## Installation & Running Locally

### Backend
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the API server:
   ```bash
   npm start
   ```

### Frontend
1. Navigate to the frontend directory:
   ```bash
   cd ../frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
