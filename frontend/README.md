# FitBot - Frontend Client

This is the React.js client interface for FitBot, styled with the premium **Sage Leaf + Soft Peach** wellness design system.

## Setup & Running Locally

1. Navigate to this directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the client dev server:
   ```bash
   npm run dev
   ```
4. Access the application in your browser at `http://localhost:5173`.
